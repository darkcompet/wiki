# Windows command line


## List

- `cd`: change directory

```bash
cd workspace
```

- `mkdir`: create new directory

```bash
mkdir project
```

- `notepad`: edit file with notepad app

```bash
notepad info.md
```

- `rm`: delete permantly a file or dir

```bash
rm old.txt
```

- `start`: open explorer

```bash
start .
``

- `type`: show content of a file

```bash
type readme.md
```
