# Grid layout


## Keyword

- Display

   ```css
   display: grid
   ```

- Gapping

   ```css
   grid-row-gap: 10px;
   grid-column-gap: 8px;

   /* or */
   grid-gap: 10px 8px;
   ```

- Merging

   ```css
   /* Make an item start from row-1 and end at row-3  */
   grid-row-start: 1;
   grid-row-end: 3

   /* Make an item start from column-1 and end at column-3  */
   grid-column-start: 1;
   grid-column-end: 3
   ```
