# PhpMyAdmin

---

## Setup

- To use phpmyadmin with server and mysql, must delcare as below:

   ```bash
   $cfg['Servers'][$i]['user'] = 'root';
   $cfg['Servers'][$i]['password'] = '1234';

   $cfg['Servers'][$i]['host'] = '127.0.0.1';
   $cfg['Servers'][$i]['port'] = '6969';
   ```
